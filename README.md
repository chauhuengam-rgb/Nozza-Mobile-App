# 🇩🇪 Nozza Mobile App - German Edition

Ein einfaches Aufgaben-Management-Tool für Android, entwickelt mit React Native und Expo. (Một ứng dụng quản lý bài tập đơn giản cho Android).

## ✨ Features
- **Localization:** Komplett auf Deutsch (Hoàn toàn bằng tiếng Đức).
- **User Interface:** Sauber und einfach zu bedienen (Giao diện sạch sẽ, dễ dùng).
- **Tech Stack:** React Native, Expo, JavaScript.

## 🚀 How to run
1. Install dependencies: `npm install`
2. Start the app: `npx expo start`
